# Omnia Produções — Site Institucional

Site institucional da Omnia Produções com foco em produção audiovisual, fotografia e programação/consultoria.

## Destaques
- Navegação com menu lateral e CTAs.
- Páginas de serviços e galerias (foto, drone, conteúdo social, institucional).
- Seções de depoimentos e contato.
- Assets de imagem, vídeo e áudio.
- Layout responsivo.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Estrutura
- `index.html` — página inicial
- `style.css` — estilos principais
- `script.js` — interações
- `servicos/` — páginas de serviços e galerias
- `login/` — área de sponsors
- `img/`, `videos/`, `audio/` — mídia

## Como rodar local
1. Abra `index.html` no navegador.
2. (Opcional) Use a extensão Live Server no VS Code.

## Deploy (GitHub Pages)
1. No GitHub: Settings > Pages.
2. Source: `Deploy from a branch`.
3. Branch: `main` e pasta `/ (root)`.
4. Salve e aguarde o link.
